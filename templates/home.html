Hello Dungeon World fans! Happy DROP TABLES Monday!

New Blank Dungeon World Character Sheet * moves not included.

After ignoring proponents of Dungeon World who wanted to write their name on a playbook sheet instead of building a character I've independantly reconsidered the design and presentation of compendium classes. I think everyone will agree my new design will be easy to use and clearly how compendium classes should have been implemented in the first place.

The Basic Idea: The first page of a compendium class contains general information and options used when creating a character. The second page is the moves list including levelup options and should be kept with the main character sheet for gameplay reference. Before this system most compendium classes took up 4 pages and did not seperate the information this way. Not many people wanted to flip thru 13-15 pages while roleplaying but hopefully 4 pages will sound more reasonable.

Massive redesign, I'm dropping the complicated class merging.
